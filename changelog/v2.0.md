# v2.0 更新日志

**发布日期**: 2026-07-08

## 🎉 重大更新

### 1. 拼图锁定 + 闪烁动画
- 拼好的拼图块自动锁定到正确位置，无法再被移动
- 锁定时触发闪烁脉冲动画效果（1秒5阶段发光）
- 所有关卡均支持锁定机制

### 2. 图片全面升级
- 普通关卡（72关）全部使用 Unsplash 高质量图片
- 6大图片分类：风景、建筑、人物、山岳、海洋、城市
- 每关名称与图片内容相符
- 动漫关卡（24关）保留 anilist.co 图片源

### 3. 难度系统重构
- 难度从14个精简为8个：3×3 ~ 10×10
- 难度标签直接使用网格尺寸（如"3×3"、"5×5"）
- 移除11×11~16×16超高难度
- 关卡总数从156关调整为96关（每难度12关：9普通+3动漫）

### 4. 退出逻辑优化
- 游戏中按返回键 → 弹出保存对话框（3个选项）：
  - 继续游戏
  - 保存并退出
  - 不保存退出
- 退出后返回选择关卡界面（非主菜单）
- 主菜单按返回键直接退出应用

### 5. 保存游戏进度
- 支持保存/恢复游戏进度（拼图块位置、锁定状态、步数、时间等）
- 30秒自动保存
- 下次进入关卡自动恢复进度
- 完成关卡后自动清除保存的进度

### 6. 关卡进度显示
- 未完成但有保存进度的关卡：显示进度条 + 百分比
- 已完成关卡：显示星数评价
- 进度条使用渐变色（绿色系）

## 🔧 技术改进
- 存储版本升级 v2 → v3（避免旧数据冲突）
- 成就系统更新：all_wins 目标改为96关
- 移除 ultimate_clear 成就（超高难度已移除）
- CSS清理：移除6个无用badge样式
- 构建环境：JDK 17 → JDK 21
