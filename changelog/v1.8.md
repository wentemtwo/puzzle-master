# v1.8 更新日志

**发布日期**: 2026-07-08

## 变更内容

### 移除国产手机触摸优化
- 移除 `touchPieceEl` 缓存机制，还原到v1.0简单触摸处理
- 移除 `touchStartTime` 和 `isSwiping` 滑动检测变量
- 移除触摸视觉反馈（拼图块按下高亮效果）
- 触摸处理还原为v1.0版本：3变量 + 20px阈值判断

### 移除国产手机拼图区域CSS适配
- `.puzzle-container` 移除 `flex: 1 1 0%`、`touch-action: none`、`overscroll-behavior: contain`
- `.puzzle-grid` 移除 `overflow: hidden`、`width: 100%`
- `.puzzle-piece` 移除 `touch-action: manipulation`、`-webkit-tap-highlight-color`、`user-select: none`、`:active` 伪类
- `.screen.active` 还原为 `display: block`（移除flex布局）
- 移除 `#gameScreen.active { overflow: hidden }`
- 媒体查询移除拼图相关国产手机适配规则

### 新增云端构建服务
- 创建 GitHub Actions 工作流配置 (`.github/workflows/build-android.yml`)
- 支持自动构建：push到main/master分支、tag推送、手动触发
- 构建环境：Ubuntu + Node.js 20 + JDK 21 + Android SDK
- 支持Debug和Release APK构建
- Release签名需配置GitHub Secrets（KEYSTORE_BASE64等）
- 自动上传artifact和创建GitHub Release

### 保留功能
- v1.7的42个动漫角色关卡（id 115-156）
- 所有关卡无需前置解锁条件（unlockStars=0）
- v1.3国产手机关卡选择界面适配（单滚动容器+sticky标签）
