# v1.0 更新详情 (2026-07-07)

## 首发功能

### 游戏核心
- 114个关卡，14个难度级别
  - 简单 3×3 (8关) → 中等 4×4 (8关) → 困难 5×5 (8关) → 专家 6×6 (8关)
  - 大师 7×7 (8关) → 传奇 8×8 (8关) → 史诗 9×9 (8关) → 神话 10×10 (8关)
  - 至尊 11×11 (8关) → 混沌 12×12 (8关) → 永恒 13×13 (8关) → 创世 14×14 (8关)
  - 天命 15×15 (8关) → 终极 16×16 (8关)
- 三种操作方式：点击交换、拖拽、触摸滑动
- 可解性校验（逆序数算法），确保每局都有解

### 游戏系统
- 计时器与步数统计
- 进度条（已正确/总数）
- 连击系统（连续正确放置触发连击）
- 星级评价（1-3星，基于时间和步数）
- 关卡解锁（需累计星星数）

### 辅助功能
- 智能提示（3次，自动将错位块放到正确位置）
- 高亮提示（标记所有错误位置的块）
- 撤销功能（可回退操作）
- 预览原图（长按查看完整图片）
- 编号显示（设置中开启拼图块编号）

### 成就系统
- 18个成就：初出茅庐、小有成就、拼图达人、至高荣耀、全关制霸等
- 自动检测并弹出成就通知

### 其他
- 统计面板（总星星、完成数、最佳时间/步数、最大连击等）
- 设置面板（音效/动画/编号/主题切换）
- 音效系统（Web Audio API合成音效）
- 粒子背景动画
- 深色/浅色主题

### 平台
- PWA支持（Service Worker离线缓存、manifest.json）
- Android APK（Capacitor 7.0打包）

## 技术栈
- 纯前端：HTML + CSS + JavaScript
- 图片源：picsum.photos
- 打包：Capacitor 7.0 + Gradle 8.11.1
- 最低Android版本：API 23 (Android 6.0)
- 目标Android版本：API 34 (Android 14)
