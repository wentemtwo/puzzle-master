# v1.1 更新详情 (2026-07-08)

## 新功能

### 拼图正确位置锁定
放到正确位置的拼图块将自动锁定，防止误操作：
- 绿色边框高亮标记正确位置
- 右上角绿色圆形✓图标提示已锁定
- 禁止点击选中、拖拽、触摸滑动、交换操作
- 交换逻辑增加锁定检查，任一方已锁定则交换无效

### 边缘到边缘全屏显示
适配Android导航栏和状态栏，实现沉浸式全屏体验：
- 透明状态栏和导航栏
- 暗色系统栏图标（适配深色主题）
- 刘海屏适配（displayCutoutMode=shortEdges）
- CSS safe-area-inset 确保内容不被系统栏遮挡

## 优化

### 移动端UI
- 按钮最小触摸区域 44px（符合Apple HIG标准）
- 游戏标题、副标题字号缩小适配手机屏幕
- 菜单内容flex居中布局
- 拼图区域flex:1自适应，不同网格尺寸均居中显示
- 游戏工具栏底部safe-area-inset padding

### 触摸体验
- touch-action: manipulation（消除300ms延迟）
- -webkit-tap-highlight-color: transparent（去除点击高亮）
- -webkit-user-select: none（防止误选文字）
- overscroll-behavior: none（防止过度滚动）

## 修复
- Android导航栏遮挡游戏内容
- 状态栏颜色与深色主题不协调
